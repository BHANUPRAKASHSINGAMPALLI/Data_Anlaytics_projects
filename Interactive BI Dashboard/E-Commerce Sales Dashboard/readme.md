# 📊 E-Commerce Sales Dashboard (Power BI)

## 📌 Problem Statement
Built an interactive Power BI dashboard analyzing 541,909
e-commerce transactions to track KPIs, customer behavior,
and product performance for a UK-based online retailer.

## 🛠️ Tools & Technologies
- Power BI Desktop, Power Query, DAX
- Excel for initial data cleaning

## 📊 Dashboard Pages
Page 1 — Executive Overview (Revenue, Orders, Customers KPIs)
Page 2 — Product Analysis (Top SKUs, Category Performance)
Page 3 — Customer Segmentation (RFM-based segments)

## 🔍 KPIs Tracked
- Total Revenue | Total Orders | Avg Order Value
- Monthly Revenue Trend | Top 10 Products
- Customer Retention Rate | New vs Returning

## 📐 DAX Measures Used
- Total Revenue = SUM(Sales[Revenue])
- Avg Order Value = DIVIDE([Total Revenue], [Total Orders])
- MoM Growth = (Current - Previous) / Previous * 100
