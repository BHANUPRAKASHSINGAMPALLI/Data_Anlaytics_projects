# 🛒 Superstore Sales Analysis

## 📌 Problem Statement
Analyzed 9,994 sales transactions to identify revenue trends,
top-performing products, and regional performance gaps to
support data-driven business decisions.

## 🛠️ Tools & Technologies
- Python, Pandas, Matplotlib, Seaborn
- Jupyter Notebook

## 📊 Key Questions Answered
1. Which months generate the highest revenue?
2. Which product categories drive the most profit?
3. Which regions are underperforming?
4. What is the average order value by segment?

## 🔍 Process
1. Data Cleaning → handled nulls, fixed date formats
2. EDA → trends, distributions, correlations
3. Visualization → charts for each business question
4. Insights → actionable findings documented

## 📈 Key Findings
Peak revenue month - 2017-11 ($118,448)

Top product by profit - Canon imageCLASS 2200 Advanced Copier 

Best performing region - West 

Technology profit share - $145,455  (50.8% of total profit) 

Discount-profit correlation - -0.219
